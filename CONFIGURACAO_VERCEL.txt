CONFIGURAÇÃO DAS VARIÁVEIS DE AMBIENTE NO VERCEL
================================================

Acesse: https://vercel.com/dashboard → Seu Projeto → Settings → Environment Variables

Configure as seguintes variáveis:

1. Autenticação Admin:
   AUTH_SECRET=gere_uma_string_aleatoria_de_32_caracteres_aqui
   AUTH_SALT=gere_outra_string_aleatoria_aqui
   ADMIN_USER_1=tamaraleal
   ADMIN_PASS_1=$2b$10$UNUme80JKfoNaL/HjJZ6iO2Tjg7.m//1tJ1Wwg5oz2IWwrAJF.chq
   ADMIN_USER_2=johnnyhelder
   ADMIN_PASS_2=$2b$10$UNUme80JKfoNaL/HjJZ6iO2Tjg7.m//1tJ1Wwg5oz2IWwrAJF.chq

2. Supabase (já configuradas):
   NEXT_PUBLIC_SUPABASE_URL=(já configurada)
   NEXT_PUBLIC_SUPABASE_ANON_KEY=(já configurada)

3. Email (já configurada):
   RESEND_API_KEY=(já configurada)

IMPORTANTE:
- Os valores ADMIN_PASS_1 e ADMIN_PASS_2 são hashes bcrypt da senha "New***159753"
- Para gerar novos hashes: node generate-bcrypt-hash.js "SuaNovaSenha"
- AUTH_SECRET deve ser uma string aleatória segura (ex: openssl rand -hex 32)
- Após configurar, faça um redeploy para aplicar as mudanças
